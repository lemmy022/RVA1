MAME stands for Multiple Arcade Machine Emulator. When used in conjunction with images
of the original arcade game's ROM and disk data, MAME attempts to reproduce that game as
faithfully as possible on a more modern general-purpose computer. MAME can currently emulate
several thousand different classic arcade video games from the late 1970s through the modern era.

Instructions

After you download the file all you need to do is just extract it from zip. In the fonder that
you extracted you will find mame32.exe, just start it. It will show you which games are available
for playing, double click on the one you want to play and it will launch. For setting up the
controls simply hit the TAB key.
